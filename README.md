# Low Level Design (LLD) Repository

A collection of Low Level Design (LLD) implementations built using Object-Oriented Design (OOD) principles and design patterns. This repository is intended for learning, interview preparation, and understanding scalable software design.

---

# Objectives

- Learn Object-Oriented Design principles
- Implement real-world system designs
- Apply SOLID principles
- Use appropriate Design Patterns
- Improve code maintainability and extensibility
- Prepare for Software Engineer/System Design interviews

---

# Tech Stack

- Language: Java (or C++ / Python)
- Build Tool: Maven / Gradle
- Testing: JUnit
- Version Control: Git

# Coding Guidelines

- Follow SOLID principles
- Prefer Interfaces over Concrete Classes
- Keep classes Single Responsibility
- Avoid God Objects
- Use Dependency Injection
- Write Unit Tests
- Keep methods small and readable

# Running the Project

Clone repository

```bash
git clone https://github.com/<username>/lld.git
```

Go to project

```bash
cd parking-lot
```

Build

```bash
mvn clean install
```

Run

```bash
mvn exec:java
```

Run Tests

```bash
mvn test
```
